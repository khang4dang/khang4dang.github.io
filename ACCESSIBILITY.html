Accessibility

Accessibility choices included in this static website package.

Included

  • Skip link to main content.
  • Semantic headings and landmarks.
  • Keyboard-accessible dropdowns via hover and focus-within.
  • Visible focus states.
  • Reduced-motion support for visitors who prefer reduced motion.
  • No information conveyed only by color.

When adding images or videos

  • Write meaningful alt text for informative images.
  • Use empty alt text only for decorative images.
  • Add captions or transcripts for videos where possible.
  • Avoid autoplaying audio.
