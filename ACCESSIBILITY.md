# Accessibility Notes

This website package is designed to be screen-reader- and keyboard-friendly.

## Included

- Skip link on every page.
- Semantic `header`, `nav`, `main`, and `footer` regions.
- Keyboard-accessible navigation dropdowns using focus-within.
- Visible focus states.
- Reduced-motion handling for users who prefer reduced motion.
- Cards use real links rather than clickable divs where possible.
- Gallery cards are keyboard activatable and open a dialog with a clear close button.

## When editing

- Add meaningful `alt` text to every informative image.
- Use captions/transcripts for videos when possible.
- Avoid autoplaying sound.
- Keep headings in order: one page topic, then sections.
- Do not rely only on color to show status or category.
